# composite 合成评分

## 1. 问题

三个独立维度（structural / complexity / fragility）各自给出 0-100 分数，但用户需要一个综合的项目健康分来快速判断整体状态。需要将三个维度加权合成为一个 composite 分数，并统一输出格式。

## 2. 目标用户

使用 decay 的 AI agent。agent 运行 `decay` 后获得一个综合健康分和三个维度明细，用于决定是否需要介入。

## 3. 核心假设

**三个维度加权合成 composite 分数 → 用户一眼判断项目健康状态，不需要分别解读三个维度。**

验证方式：运行 `decay`，输出包含 composite: XX/100 和三个维度明细分数。

## 4. 方案

- **Before**: 三个独立分数需要分别解读 → **After**: 一个 composite 分 + 三维度明细，一目了然

## 5. 验收标准

- 运行 `decay` → 输出包含 composite 分数（0-100）和三个维度分数
- composite = 三个维度的加权平均（权重可调）
- 某个维度为 N/A 时 → 该维度不参与合成，权重重新分配
- 输出格式清晰：composite 分数 + 各维度明细

## 6. 排除项

- 不包含评分等级映射（A/B/C/D → M6 输出格式）
- 不包含问题诊断和重构建议（→ M4）
- 不包含历史对比和趋势（→ M5）
