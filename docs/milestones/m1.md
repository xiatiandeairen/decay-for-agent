# M1 — 交付 v0.1.0 本地功能闭环

## 1. Goal

### Definition of Done

`decay` 可以在本地 Rust 项目中完成函数扫描、保存命名 baseline、比较 baseline 与当前工作区、比较两个 baseline，并提供当前风险体检。

### Value Target

作者可以在 commit 前获得函数级复杂度退化证据，而不是只依赖人工读 diff 或 AI 自评。

### Key Hypothesis

- **Hypothesis**: Rust 函数级复杂度退化可以通过 baseline + active metrics + diff 在本地 CLI 中形成可解释证据链。
- **Type**: technical feasibility
- **Validation method**: 在 fixture 和当前项目中运行测试、clippy、`decay`、`decay --help`、`decay doctor`；功能通过不等于产品价值成立。
